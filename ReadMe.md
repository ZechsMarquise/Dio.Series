## Description
Creating an APP to register series in memory

## Usage
### Version 1
### Apresentação do menu

DIO Series <3
Escolha uma opção abaixo:
1- Listar séries
2- Inserir nova série
3- Atualizar série
4- Excluir série
5- Visualizar série
C- Limpar Tela
X- Sair
